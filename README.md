# Lanspire

## Main features

- Course management
- Class management
- Student management
- Staff management
- Statistics
- Settings, regulations

## Tech stack

- Client: React, Ant Design
- State management: React Redux
- Server: Node.js, Express, Sequelize ORM
- Database: PostgreSQL, Firebase Storage (for assets)
